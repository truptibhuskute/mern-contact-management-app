# MERN Contact Management App

This is a simple Contact Management web application built using the MERN stack.

## Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- State Management: useState

## Features
- Add new contacts with validation
- View stored contacts without page reload
- REST APIs for create and fetch contacts
- Responsive UI
- Publicly deployed application

## Live Demo
Frontend: https://your-frontend-url.vercel.app  
Backend: https://your-backend-url.onrender.com

## Setup
1. Clone the repo
2. Install dependencies for frontend and backend
3. Add MongoDB URI in backend `.env`
4. Run backend and frontend separately
